# FineVLA Project Page

This is the project page for the paper:

**FineVLA: Fine-Grained Instruction Alignment for Steerable Vision-Language-Action Policies**

## Setup

To deploy on GitHub Pages, enable Pages in your repository settings and set the source to the `main` branch root (`/`).

## Adding Assets

Place your assets in the following paths:

- `static/images/teaser.png` — Overview / teaser figure
- `static/images/method.png` — Method pipeline figure
- `static/images/results.png` — Results figure
- `static/videos/demo.mp4` — Demo video
- `static/pdfs/paper.pdf` — Paper PDF
